9. Downloads

9.1 How to Access the Downloads Menu

The Downloads option allows you to view your Steam Deck's latest downloads. This is where you will find the progress of any games you are downloading. You can also view any updates to your games here.

Within the Downloads menu, you can also click the A Button on any of the titles to be taken to that game's page:

9.2 Download Options

Navigating over a game and pressing the Menu Button allows you to do several things.

Play: Plays the game

Remove from List: Removes the game from the Downloads List

Add to Favorites: adds the game to your Favorites menu

Add the game to a new collection

Manage the game:

  • Hide this game
  • Mark the game as Private, hiding it everywhere but your library. (Other people cannot see that you own it or when you play it, for example.)
  • Uninstall the game
    • The Developer menu allows you to:
  • Delete the Proton Files (allows games to run compatiably on the Steam Deck)
  • Clear Selected Controller Layout: allows you to delete the saved controller setup you have mapped to the game
